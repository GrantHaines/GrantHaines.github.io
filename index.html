Skills

Programming Languages

>C/C++ Most of my early programming experience was in C and C++, and I have continued to use it since for numerous programming projects and assignments of varying scale. Notably, these include the small shell and spaceship escape game listed in my projects above.

>C# I learned C# in the course of working on the Aeternitas Station game project mentioned above. Although that is the only work I have done with it, the project was substantial in size and it is similar enough to C and C++ that I feel very confident with it.

>Python I have used Python in the creation of a large number of smaller scripts, especially for statistical analysis, but have not created any sizable programs with it.

>JavaScript My experience in JavaScript is limited mainly to web development, either with the NodeJS framework or for client-side web page interaction.

>Java I have used Java in a couple of smaller projects and have a general knowledge, but am not intimately familiar with it.

>Assembly I have some experience programming with basic assembly language on an AVR Microcontroller, and have studied the syntax of AT&T x86 assembly.

>Haskell I have spent some time learning the basic concepts of Haskell and functional programming in general, but most of my experience in using it is with small demo programs.

>Prolog Like Haskell, I have studied the basic concepts of Prolog and logical programming languages, but do not have extensive experience with using it in practice.

Hardware and Software
